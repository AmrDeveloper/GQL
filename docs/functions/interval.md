### Interval Operators

| Operator | Arguments            | Description                  |
| -------- | -------------------- | ---------------------------- |
| +        | (Interval, Interval) | addition two intervals       |
| -        | (Interval, Interval) | subtraction two intervals    |
| \*       | (Interval, Interval) | multiplication two intervals |
| /        | (Interval, Interval) | division two intervals       |
